# Parallel_Processing_Project
Communication Performance, CPU v GPU

## Micro Benchmarks
To clone the microbenchmarks along with this repo, run this command after cloning:
```
git submodule update --init --recursive
```

## Running Tests
```
mkdir build
cd build
cmake ..
make all
make tests
```
